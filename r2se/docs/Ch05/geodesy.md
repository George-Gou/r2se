# 概述

!!! abstract "导言"

    本章主要介绍大地测量工程项目。     

## GPS点标识

B级点：基岩GPS、水准公用标识；  

C级点：土层GPS、水准公用标识；  

高程异常控制点：天线墩、水准共用标识。  

## 水准测量

一、二等水准测量外业计算改正：  

1）水准标尺长度改正；2）水准标尺温度改正；3）正常水准面不平行的改正；4）重力异常改正；5）固体潮改正；6）环线闭合差改正；  

三、四等水准测量改正：  

1）水准标尺长度改正； 2）正常水准面不平行改正；3）环线闭合差改正；  

其中正常水准面的不平行改正是这样的，正常水准面不平行导致沿着不同水准线路测量结果不一致。  

#### 规划投影计算

斜距D投影到高斯平面D'，分别经过了**方向和边长的归化改正和投影改正**。其中方向的归化改正为三差改正，即垂线偏差改正、标高差改正、截面差改正。边长经过边长归化改正，它与测区平均高程到参考椭球体的大地高有关；  

投影改正与测区到中央子午线的距离有关，越远，方向和边长的投影改正值越大。

## GNSS控制网

提高网络RTK的手段：选用多频接受机、选用3个以上控制点、控制点的检校、流动站观测长时间。水准路线上注意闭合差的平差分配。
